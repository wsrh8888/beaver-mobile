# 海狸IM

### 项目介绍
+ 本项目是利用工作之余做的一个聊天IM。
+  前端使用uniapp， 后端使用go-zero， 桌面端使用electron
+ 如果喜欢可以点一个star
+ 加入QQ群：[![加入QQ群](https://img.shields.io/badge/加入QQ群-1013328597-blue.svg)](https://qm.qq.com/q/82rbf7QBzO)（1013328597）
+ 加入微信群：
+ <img src="./static/wxQcode.JPG" width="200" height="200" alt="微信群二维码"/>


### 功能介绍
+ 用户注册
+ 用户登录
+ 好友列表
+ 好友详情
+ 个人信息
+ 修改头像(七牛云)
+ 个人二维码
+ 扫码添好友
+ 私聊(文本、图片、表情)
+ ws消息
+ 删除好友
+ 网关
+ 群聊

### uniapp启动命令
```
node版本: 20
npn install
npm run build_test

执行完成在uniapp客户端启动
```
